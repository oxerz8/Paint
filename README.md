# Paint
Recreating MS Paint using Java


![Image](Screenshot.png)

## Features included:
* Draw Circles
* Draw Quadilaterals
* Draw Squiggly lines
* Draw Polylines
* Change line thickness
* Change color
* For solid shapes, fill/no fill option offered
